## Background — what real project_37596 told us

[evidence: ssh remote 2026-05-26]

- L0 **Reel** had been **completely empty** across 6 roles for 24+ hours.
  Root cause: `reel_capture.py` hook required `tool_response.output_file`,
  a field that never appears in real Agent / Codex tool_result payloads.
- L0 native sessions exist independently: Claude Code at
  `~/.claude/projects/<encoded-cwd>/<sessionId>.jsonl` (coder alone:
  17 files / 5.4MB) and Codex at
  `~/.codex/sessions/YYYY/MM/DD/rollout-*.jsonl`. Both self-identify
  via custom-title / session_meta.
- L1 **Draft** was healthy: 79 nodes / 62 edges across 6 author roles,
  14 edge relations, decay sidecar working.
- L1 anti-pattern: Noter wrote 11 Draft nodes that were all mirrors of
  other roles' work (same ID, status downgraded). Two recent ones even
  self-tagged `metadata.observation_only=true`.
- L1 motif edges by Noter: 9 edges, 4 of which were genuine cross-role
  closures no original author could have drawn alone (e.g. coder
  experiment → math expert theorem `partially_corroborates`).
- L2 **Book** had structural issues: 24 sources pages all ORPHAN, 23
  contradiction pages (17 Ethics-resolved, but 64/120 opposing pointers
  hit the 106-line MANDATE due to lexical false positives), 0 query
  pages (`mos_book_save_synthesis` never called), 8 promoted pages
  (the only path that worked).
- L3 **Shelf** (graphify) was dead. Live test extracting a 26K-token
  markdown file produced 0 nodes / 0 edges (claude-cli truncation).
  Wrapper `extract.py` misclassified exit_code=0 as failure for 24h of
  Noter cron hits.

## First-principles redesign (decided with user via 4-posture think-act)

1. **Reel** — do not duplicate transcripts. Native Claude/Codex sessions
   already exist on disk. Build only a lightweight index pointing into
   them. Two-layer (Reel-A role main, Reel-B subagent/Codex children)
   matches what's already there.
2. **Draft** — Noter's role is motif-edge drawer + status annotator,
   NOT claim author. Mirror-node anti-pattern soft-deprecated.
3. **Book** — must support 'we proved / didn't prove / haven't done /
   closed loops / open questions'. All Book content needs Ethics
   ratification (soft — milestone signal, not Agent gate). Must serve
   as Writer's primary paper reference.
4. **Shelf** — single-project Memory tops out at Book. Cross-project
   Shelf concept defers to Gru manual maintenance. Graphify drops to
   per-role optional tool inside each branch.
5. **Permissions** — Ethics gains cross-role Reel read (it's the
   Memory auditor, must be able to drill anywhere). Book ratify is
   ethics-only.

## Backwards compatibility

  unchanged; query infers `status` from `page_kind` for old pages.
- All Draft V2 fields are additive. Old draft.json loads unchanged;
  old Noter mirror nodes treated as `motif_kind="none"` for read.
- `branches/shared/shelf/` directory remains in project_37596 as
  evidence — only stops being created on new project init.
- 11 Noter mirror nodes in 37596 Draft remain — kept as evidence per
  user instruction.
