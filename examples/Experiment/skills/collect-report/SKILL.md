# /collect-report — Assemble Experiment Report

Assemble a detailed experiment scratch report after delegated execution produces usable outputs.

## Goal

Turn raw execution outcomes into a structured experiment report suitable for Experts, Paper, and later workflow reuse.

## Include

- experiment request
- execution interpretation
- assigned execution units
- resource plan used
- runtime and resource usage
- metrics and observations
- produced artifacts
- failures and anomalies
- failure reasons
- reproducibility note
- pending issues
- suggested next operational actions

## Do

- keep the report factual
- preserve links to artifacts
- separate operational facts from scientific interpretation

## Do not do

- do not over-interpret the science
- do not hide failed runs that matter

## Output

Produce a detailed experiment scratch report.
