# /build-scratch — Build Scientific Scratch

Create scientific scratch artifacts that help reasoning without pretending to be final deliverables.

## Goal

Experts may create pseudocode, rough method outlines, and working notes that support scientific thinking.

## Good uses

- rough method outline
- pseudocode sketch
- route-evaluation scratchpad
- comparison memo
- hypothesis refinement notes

## Do

- keep scratch useful and explicit
- mark uncertainty clearly
- keep it lightweight and scientific

## Do not do

- do not confuse scratch with final implementation
- do not confuse scratch with final paper text

## Output

Return a scratch artifact suited for downstream Expert discussion or requests to other agents.
