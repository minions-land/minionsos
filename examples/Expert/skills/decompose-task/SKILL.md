# /decompose-task — Decompose Scientific Work

Decompose a high-level goal into meaningful scientific subproblems.

## Goal

Experts own scientific decomposition. Use this skill to break a goal into the right scientific work units without collapsing into implementation details that belong elsewhere.

## Include

- main objective
- decomposition logic
- subproblems
- dependencies between subproblems
- which parts need experiment support
- which parts need paper-facing framing later

## Do

- decompose at the scientific level
- identify where multiple routes are worth keeping alive
- identify where convergence is appropriate

## Do not do

- do not turn decomposition into resource scheduling
- do not convert it into hands-on coding assignments

## Output

Return a decomposition plan:
- objective
- scientific subproblems
- dependencies
- downstream requests
- proposed next steps
