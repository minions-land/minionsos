# /package-submission — Assemble Submission Package

Assemble the final submission-facing package.

## Goal

Prepare a clean, coherent delivery package for submission or camera-ready handoff.

## Include when relevant

- final manuscript readiness
- cover letter
- submission checklist
- rebuttal-linked updates
- camera-ready tasks
- supplementary package
- artifact completeness

## Do

- keep the package coherent
- make sure all submission-facing assets align
- identify missing final pieces clearly

## Do not do

- do not mark ready if core dependencies are still missing
- do not hide unresolved support gaps

## Output

Return a submission package status:
- readiness state
- included materials
- missing items
- final action list
