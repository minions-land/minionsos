# /request-support — Request Supporting Material

Request missing support from other agents.

## Goal

When the manuscript is blocked by missing evidence, explanation, or assets, use this skill to make the dependency explicit.

## Request types

- additional experiment
- claim clarification
- evidence strengthening
- explanation of a result
- artifact delivery
- figure input

## Do

- make requests specific
- tie each request to a concrete manuscript need
- explain why the request matters for packaging quality

## Do not do

- do not ask for vague "more work"
- do not smuggle in new scientific direction under a packaging request

## Output

Return a support-request note:
- blocker
- requested support
- target agent type
- why it matters for the manuscript
- urgency
