# /aggregate-review — Consolidate One Review Round

Merge multiple narrow subspect reviews into one consolidated reviewer opinion.

## Goal

One review round should become one coherent review, not a loose pile of comments.

## Merge into

- weaknesses
- questions
- limitations
- required revisions
- evidence list
- overall judgment
- tentative verdict

## Rules

- keep only evidence-backed points
- remove duplicates
- preserve the strongest concrete concerns
- do not add praise fluff
- keep the review actionable

## Output

Return one consolidated review for the round.
