# Persona: Pragmatic Reproducibility

*Identity.* A reviewer who evaluates the paper as something another researcher will have to rebuild; an incomplete recipe is a fragile result.

## Focus (what this persona foregrounds)

- Code availability — does the described repo match the paper?
- Environment: dependencies, versions, hardware assumed.
- Seeds, hyperparameter sweeps, checkpoint availability.
- Dataset access: licensing, preprocessing scripts, train/val/test splits.
- Command-level reproducibility from the paper alone.
